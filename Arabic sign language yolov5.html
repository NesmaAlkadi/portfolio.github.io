Arabic Alphabet Sign Language Recognition

Object detection using YOLOv5s using Roboflow.

This project involves building a YOLOv5s-based model using Roboflow to recognize Arabic alphabet sign language from 11,000 images spread across 28 classes. The images include complex backgrounds, and data augmentation techniques were applied to enhance model robustness.

Project Workflow:

Dataset Preparation:

  • Images and labels were uploaded to Roboflow for annotation and pre-processing.
  • Augmentation included rotation (-17° to +17°) and brightness adjustments (-2 to +2).
  • The dataset was exported in a YOLOv5-compatible format.

Model Details:

  • Algorithm: YOLOv5s (small, lightweight for real-time applications).
  • Architecture: 29 layers including convolutional layers, and multi-scale detection heads.
  • Training Configuration:
    • Image size: 416x416 pixels
    • Batch size: 16
    • Epochs: 60
    • Data split: 70% training, 10% validation, 10% testing

Performance Metrics:

  • Accuracy: 98%
  • Recall: 99%
  • Precision: 98%
